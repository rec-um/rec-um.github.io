REC Class

Learn from basic, start from zero

REC Class is a series of lectures conducted by REC Engineering Team annually, aimed to provide first year students the knowledge of Arduino. Topics ranging from lighting up an LED, to turning a motor, to register programming etc. with all time favorite Arduino Uno.

Previ​ous REC Class (2020)
Time: 2 hours per week, 8 weeks

Fee: NONE

platform: MICROSOFT TEAM

Participants: first year ee students & others​
After rec class, we learnt
- Arduino basic
- arduino wiring
- arduino coding
- input and output
- servo motor
- arduino register
PCB Workshop
through the via, route the trace

PCB Workshop was a 3-days workshops conducted by REC PCB Engineering Team. Participants got to hands-on with schematic and printed circuit board (PCB) design using Altium software. At the end of the workshop, everyone designed their own stream deck. 

PCB Workshop

20 - 22 August 2021
What had been learnt:
- PCB Basic
- Getting started with Altium Software
- Schematics designing
- Schematics to PCB conversion
- PCB Routing
- Designing own Stream Deck PCB board
